# 🏠 Dorm Door
##Introduction:

Dorm Door is a simple dorm booking website for students. It lets anyone browse dorms and rooms, check availability, compare facilities and prices, and submit a booking/application form without creating an account. After applying, students receive a unique reference ID to track their status (pending/approved/waitlisted) anytime. On the admin side, hostel authorities can manage rooms and availability, review applications, approve allocations, and send updates efficiently.
