🏠 Dorm-Door Platform

📌 Project Overview

Dorm-Door Platform is a MERN stack web application designed to connect students with dormitory administrators to simplify accommodation management.

Students often struggle to find suitable dormitories and communicate with dorm managers. This platform provides a centralized system where students can easily browse available dorms, send accommodation requests, and communicate with administrators.

Dorm administrators can manage dorm listings, update room availability, and respond to student requests efficiently.
The platform aims to make the student housing process more transparent, organized, and efficient.

✨ Key Features
👨‍🎓 Student Features

1. View available dormitories
2. Search and filter dorm options
3. View dorm facilities and details
4. Send accommodation requests
5. Track request status

🏢 Dorm Administrator Features

1. Add and manage dorm listings
2. Update room availability
3. Approve or reject accommodation requests
4. Manage dorm information

🔐 System Features

1. Secure authentication system
2. Role-based access control (Student/Admin)
3. Responsive user interface
4. RESTful API integration
